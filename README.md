This repository contains implementations of core **compiler design concepts**
including lexical analysis and predictive parsing using **Lex/Flex and C++**.

## Projects

### 1. LL(1) Predictive Parser
- Implements a **LEX-based tokenizer**
- Builds an **LL(1) predictive parser**
- Constructs a **binary parse tree**
- Evaluates arithmetic expressions

### 2. LaTeX Lexical Analyzer
- Detects **LaTeX commands, environments, and math formulas**
- Uses **Lex regular expressions**
- Maintains statistics using custom symbol tables

## Technologies
- C++
- Lex / Flex
- Compiler Design Concepts
